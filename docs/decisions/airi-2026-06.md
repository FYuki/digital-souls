# AIRI採用方針 (2026-06)

## 背景

Phase 3 でコア基盤の検証に入るにあたり、人格・記憶・エージェント制御を担うフレームワークの方針を固める必要があった。

候補として AIRI を検討してきたが、今回この方向性を正式決定とする。

---

## 決定内容

**AIRIを、人格・記憶・エージェント制御のコアフレームワークとして採用する。**

---

## 保留事項

**フォークするか・外部接続で使うかは Phase 3 で判断する。**

Phase 3 の検証結果（カスタマイズ範囲・接続コスト・保守性）を見てから決める。
現段階では採用方針のみ確定し、統合方式は未確定とする。

---

## 根拠

- AIRI はパーソナルAI向けの設計思想を持つフレームワークであり、光織のような長期パートナー型AIの要件に適合する
- 人格・記憶・エージェント制御を一体で扱える設計が、このプロジェクトの方向性と一致する
- Phase 3 で本格検証するまでコードには触れない方針のため、今回は採用方針の確定にとどめる
